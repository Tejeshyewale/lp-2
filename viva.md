An expert system used to manage employee or organizational data efficiently.

## 2. What are the features of this system?

* Add records
* Search records
* Delete records
* View employee database
* Analytics dashboard

## 3. Which technology is used in this project?

Pandas is used to store and display employee records in table format.

Rule-based AI.

## 8. What are the advantages of this system?

* Fast record management
* Easy searching
* User-friendly interface

# Common Expert System Theory Questions

## 1. What is an Expert System?

An expert system is an AI application that solves problems like a human expert.

## 2. What are the main components of an Expert System?

* Knowledge Base
* Inference Engine
* User Interface

## 3. What is Knowledge Base?

It stores rules and facts.

## 4. What is Inference Engine?

It applies rules to generate decisions.

## 5. What is Artificial Intelligence?

AI is the simulation of human intelligence using machines.

## 6. What is Rule-Based System?

A system that uses IF-THEN rules.

## 7. What is Forward Chaining?

Starting from facts to reach conclusions.

## 8. What is Backward Chaining?

Starting from goals and checking facts.

## 9. Advantages of Expert Systems?

* Fast decisions
* Reduced errors
* Easy automation

## 10. Limitations of Expert Systems?

* Cannot learn automatically
* Depends on predefined rules

## 11. What is Streamlit?

A Python framework used to build web applications.

## 12. Difference between Tkinter and Streamlit?

Tkinter creates desktop GUI while Streamlit creates web GUI.

## 13. What is GUI?

Graphical User Interface.

## 14. Why is Python used for AI?

Because of simple syntax and large libraries.

## 15. Applications of Expert Systems?

* Healthcare
* Banking
* Airlines
* Help desk
* Trading systems
